This is the .NET implementation of Fit.

To run Fit, use the following command line:

runfile input.html output.html dll-path

For example, running fhe following command from the Fit installation directory will process the Fit arithmetic example and create a "results.html" file in the current directory (don't forget the dot at the end):

runfile source/examples/arithmetic.html results.html .

For information about Fit, see the Fit website.
