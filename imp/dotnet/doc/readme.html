To Run This Version of Fit

Try it now: open a command prompt and change to the Fit installation directory. Run the following command. It will process the arithmetic example and put the output in results.html. Don't forget the dot at the end:

runfile source/examples/arithmetic.html results.html .

You'll see this message:

37 right, 10 wrong, 0 ignored, 2 exceptions

To use Fit on your own project, make sure runFile.exe and fit.dll are in your path and use the following command-line:

runfile input.html output.html dll-path

Put the directory that contains your fixture DLL in "dll-path." You can include more than one directory if you separate them with semicolons. Run "runfile" without any arguments for more information.

See the Fit website for more information about Fit.

Archive Contents

In this archive, you will find the following files and directories:

  • runFile.exe: The Fit command-line tool.
  • fit.dll: The Fit library.
  • license.txt: The Fit license. Fit is licensed under the GPL.
  • readme.html: This file.

  • eg.dll: Fixture library for the examples.
  • fat.dll: Fixture library for the Fit Specification.
  • runWiki.exe: A variant command-line tool for using Fit on documents that have a tag around the tables to process. (You can ignore this file.)

  • examples\input\: Sample files that you can run through Fit.
  • examples\output\: What a few sample files look like after they've been run through Fit.
  • examples\eg\: The source code for some example fixtures.

  • spec\input\: The Fit Specification.
  • spec\output\: The Fit Specification after being run on this release.
  • spec\fat\: Source code for the fixtures used in the Fit Specification.

  • doc\: Supporting documentation. There's not much here—see the Fit website for the real Fit documentation.
  • source\: The source tree used to build this release.
