This is the Java implementation of Fit.

To run Fit, use the following command line:

java -classpath fit.jar fit.FileRunner input.html output.html

For example, running the following command from the Fit installation directory will process the Fit arithmetic example and create a "results.html" file in the current directory:

java -classpath fit.jar fit.FileRunner source/examples/arithmetic.html results.html

For information about Fit, see the Fit website.
